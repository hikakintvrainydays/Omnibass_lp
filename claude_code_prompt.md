## タスク：アニメーションをもっと派手に！

### 1. Our Purpose - 泡アニメーション強化
**現状:** 泡が下から上に浮かぶアニメーション（opacity 0.3）
**要件:**
- 泡のサイズをさらに大きく（20-30px）
- 透明度は維持しつつ、もっと目立つように（グローエフェクト追加）
- 数を増やす（5個 → 8-10個）
- 速度の変化をつける（一部速く、一部遅く）

### 2. サービスカード - 波紋エフェクト強化（カード #1, #2）
**現状:** ホバー時に600pxの波紋が広がる
**要件:**
- 波紋サイズをもっと大きく（800px以上）
- 複数の波紋レイヤー（二重、三重の波紋）
- 色をもっと鮮やか（ブルー系のグラデーション）
- パルスエフェクト追加

### 3. サービスカード - 釣り上げアニメーション強化（カード #3）
**現状:** ホバー時に-50px上昇、10deg回転、scale 1.08
**要件:**
- 上昇をもっと大きく（-80px以上）
- 回転をもっとダイナミックに（-15deg）
- バウンス感を追加（cubic-bezier調整）
- 影をもっと大きく派手に

### 4. PC版レイアウト拡大
**現状:** max-width: 1300px, gap: 3rem
**要件:**
- max-width を 1400px に拡大
- gap を 4rem に増加
- padding も調整（2rem → 3rem）

## 実装の注意点
- モバイル版（@media max-width: 768px）では複雑なアニメーションを無効化
- パフォーマンスを考慮（transform, opacity のみ使用）
- スクロール時のバブル表示機能（IntersectionObserver）は維持
- 既存のclass名は変更しない

- デスクトップで見たときに「うわっ！派手！」と感じるレベル
- アニメーションが滑らかで目を引く
